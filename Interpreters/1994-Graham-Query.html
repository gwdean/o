1 A Query Interpreter

The central function in this code is interpret-query, which recursively works through the structure of a complex query, generating bindings in the process.

A sample usage of the Interpreter looks something like this: 

(lookup 'painter '(?x ?y english))

(interpret-query '(and (painter ?x ?y ?z)
                       (dates ?x 1697 ?w)))
